EARLY DETECTION & CLASSIFICATION OF CANCER CELLS.

The Goal

Our primary objective was to design and develop a cutting-edge machine learning model with the capability to discern and classify cancerous cells with an accuracy surpassing 85%

The Role

In this role, I was tasked with training a Python-based model to proficiently categorise each image based on cell type, seamlessly integrating it with the binary classification model

Responsibilities

Model Selection and Testing:

Rigorously assess and identify the most suitable machine learning model for the specific use case

Algorithm Development:

Formulate an effective algorithm tailored to efficiently process a vast dataset of images for rigorous testing

Utilisation of Evaluation Techniques:

Implement advanced methodologies, including Train-Test Splits, Cross-Validation, Confusion Matrix, and Learning Curves, to meticulously analyse and enhance the model's accuracy

Choosing the Sequential model in TensorFlow mirrors Lato's modular design, with each layer serving a distinct purpose in building the neural network. The initial layer flattens the input image, akin to Lato's separate modules.

The second layer, like the Lead Front-end Developer, employs a sigmoid activation function to learn intricate patterns. This streamlined approach aligns with the specific needs of the model, much like the developer shaping the UVUX of the admin dashboard with custom themes, a flex grid system, form validation, and dynamic themes.
